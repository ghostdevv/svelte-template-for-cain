# Svelte Template
Made for cain

# Features

-   Vite
-   TypeScript
-   Routify
-   Postcss (Autoprefixer)

# Commands

Run the command with `npm run <command>`, for example `npm run dev`.<br />

| Command | Description        |
| ------- | ------------------ |
| Dev     | Run the dev server |
| Build   | Build the site     |
| Preview | Preview the site   |
